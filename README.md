# Food Delivery

## A mock website for online ordering for a restaurant

### Technologies
- ASP.Net Core
- Entity Framework
- C#
- DataTables.net
- TinyMCE

### User Types
- Administrator
- Customer

### User Functions
#### Administrator
- CRUD Users
- CRUD Food Types
- CRUD Food Categories
- CRUD Menu Items

#### Customers
- Register account
- Login
- View Menu Items
- Add/Remove Menu Items to/from Cart
- Purchase Items
